## Why This Blog Exists

I wanted a blog that feels different from the thousands of minimalist blogs on the internet.

Instead of soft shadows and rounded corners, this one embraces a **Neo-Brutalist** design language with:

- Heavy borders
- Bold typography
- Bright accent colors
- Chunky shadows
- Grid backgrounds

## Writing MDX

MDX lets you mix Markdown with React components.

Inline code looks like this:

`npm run dev`

And here's a JavaScript example:

```javascript
console.log("Hello Priyadharshan!");
```

Or a TypeScript example:

```ts
interface User {
  name: string;
  role: string;
}

const user: User = {
  name: "Priya Dharshan",
  role: "Developer",
};

console.log(user);
```
